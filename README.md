
# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

## Features

- Create memory cards with text, date, and optional image
- Typewriter composer with live character count and image preview/clear
- Automatic placement near viewport center with subtle jitter and typewriter reveal effect
- Drag to pan canvas; drag cards to reposition; click to select
- Link cards with curved connections; toggle link mode; add/remove connections
- Zoom in/out and reset view; status overlay indicating current mode
- Delete selected card and clear all (with confirmation)
- Export the entire board snapshot to PDF (powered by `html2canvas` and `jsPDF`)

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`
